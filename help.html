Help & FAQ

How to manage RFID badges in BTicino Sfera .prj files

  • What is FobForge?
    FobForge is a free, browser-based tool that lets you add, edit, and remove RFID badges and contacts in BTicino Sfera .prj files — entirely in your browser, with no installation and no server. Your files never leave your device.
  • Which BTicino systems does FobForge support?
    FobForge supports BTicino Sfera New and Sfera Robur door entry systems — the systems managed by the TiSferaDesign software, which produces .prj project files.
  • What is a BTicino .prj file?
    A .prj file is the project file created by BTicino's TiSferaDesign software. It stores all configuration for a Sfera door entry system: contacts, apartments, RFID badge assignments, and access codes. Internally it is a double-encrypted ZIP archive containing a SQLite database. FobForge can read, modify, and re-save these files while preserving all original data.
  • Why can't I add RFID badges remotely with TiSferaDesign?
    TiSferaDesign requires a USB connection to the physical reader module to register new RFID badges. This forces technicians to visit the installation site every time a new badge needs to be programmed. FobForge bypasses this limitation by editing the .prj file directly — the badge UID (printed on every transponder) is all you need.
  • How do I add an RFID badge in FobForge?
    1. Open FobForge.
    2. Load your .prj file using the Open button.
    3. Select the contact you want to assign the badge to.
    4. Click Add badge and enter the 8-character UID (printed on the badge or fob).
    5. Click Save to download the modified .prj file.
    6. Load the saved file back into TiSferaDesign or upload it to the reader.
  • How do I find the UID of my RFID badge or fob?
    The UID is usually printed on the badge or key fob as an 8-character hexadecimal code (e.g. A3F2B801). On some 125 kHz EM4102 transponders it may be printed as a decimal number — convert it to hexadecimal to use it in FobForge. On 13.56 MHz MIFARE cards the UID is typically printed on the back. You can also read the UID with any NFC-capable Android phone using a free UID reader app from the Play Store.
  • What RFID / NFC formats are supported?
    BTicino Sfera readers typically accept 125 kHz EM4102 transponders and 13.56 MHz MIFARE Classic cards. FobForge stores the UID as an 8-character uppercase hex string, which matches both formats. The exact format accepted depends on your physical reader module — consult your BTicino installation documentation.
  • Is my data safe? Does FobForge upload my files?
    Yes — your data is completely safe. FobForge runs 100% in your browser using WebAssembly. Your .prj files are never uploaded to any server; all processing happens locally on your device. The source code is open and auditable on GitHub.
  • Does FobForge work offline?
    Yes. Once the page has loaded, FobForge works entirely offline. You can open .prj files, make changes, and save the result without any internet connection.
  • Can I use FobForge instead of TiSferaDesign?
    FobForge is a complement to TiSferaDesign, not a full replacement. It focuses on the specific task of managing RFID badges and contacts in .prj files without requiring physical access to the reader. For initial system commissioning, firmware updates, or advanced configuration, TiSferaDesign with hardware access is still required.
